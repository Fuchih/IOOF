# Member Generator

## Getting Started ✨ :sparkles:

### Install dependencies

1. Front-end

```bash
$ cd .\client\
$ npm install
```

2. Back-end

```bash
$ cd .\server\
$ npm install
```

### Start the server side

```bash
$ npm run start
```

### Start the client side

```bash
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
Open console to see the result.
